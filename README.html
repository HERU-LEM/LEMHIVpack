LEMpack

LEMpack is an R package with necessary code and data to run a dynamic compartmental simulation model for the localized HIV microepidemics in Atlanta, Baltimore, Los Angeles, Miami, NYC, Seattle.

LEMpack the development of the LEM model is detailed in the following manuscripts:
